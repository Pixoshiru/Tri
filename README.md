Tri is a "ternary" numeral system, where each "bit" can carry the value 0, 1 or 2, represented visually as groups of dots that can either be on or off.
Note: In a way, it's a ternary numeral system that uses groups of 3 binary bits as its own units.

Made by Matthieu 'pixoshiru' Gallais back in the olden days of 2017 for the game Deus Ex: The Breach, at Eidos Montreal.
